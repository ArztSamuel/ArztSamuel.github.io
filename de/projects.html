HTML/CSS/JS

HTML (Hypertext Markup Language) ist eine textbasierte Sprache zur strukturierung digitaler Dokumente. HTML-Dokumente sind die Basis des World Wide Web.

CSS (Cascading Style Sheets) ist eine Sprache zur Erstellung von Gestaltungsanweisungen für HTML.

JavaScript ist eine weitere Programmiersprache. Zusammen mit HTML und CSS ist sie eine der drei essentielle Technologien des Webs.
