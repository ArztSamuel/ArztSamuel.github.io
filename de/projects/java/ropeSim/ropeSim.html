Seil Physik Simulation

Eine Physik-Simulation von 2D Seilen.

Beschreibung

Eine physikalisch korrekte Simulation eines vom Benutzer "gezeichneten" Seils unter dem Einfluss der Schwerkraft.

Steuerung:

Seile können an ihren Anfangs- und Endpunkten durch einen Linksklick gezogen werden.

Durch ziehen der geklickten rechten Maustaste können neue Seile erzeugt werden. Durch klicken der rechten Maustaste werden "Pins" erzeugt, an welchen die Anfangs- und Endpunkte der Seile hängen bleiben.

Wenn man die Steuerungstaste gedrückt hält, kann man mit dem Ziehen der linken Maustaste Seile zerschneiden. Das Programm kann durch die Rücktaste zurückgesetzt werden.
