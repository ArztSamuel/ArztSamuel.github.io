CNL - Changelog

Here you can find details to all changes made in updates to CNL

  • 1.2
  • 1.1

1.2

  • Added autmatic Updater.
    • CNL will now automatically notify users of updates and update automatically if the user agrees.
    • You can set the update check interval in CNLOptions.
    • All data required for updater is saved in a seperate file called "updater.save" in the same directory as the login data is saved.
  • Minor changes to certificate checking

This update was classified as "recommended".
