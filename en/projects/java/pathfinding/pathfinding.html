Pathfinding using the A*-Algorithm

Description

I developed this program for my special subject in computer sciences of my school leaving examinations

It visually illustrates the step by step processes of the A*-Algorithm. The goal of this algorithm is to find the shortest possible Path from A to B in a tile-map.

Instructions

You can switch between two modes with the button in the top left corner. Depending on the current mode a left mouseclick will either set new walls or reset the start, while a right mouseclick will either delete walls or reset the destination.

You can either watch A* unfold step-by-step by repeatedly pressing the spacebar or jump right to the end by pressing enter.
